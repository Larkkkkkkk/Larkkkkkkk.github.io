本地项目上传Github

前言

今天想来写一篇博客记录如何将项目通过git上传到github上

安装git

Git for window下载地址

http://www.xp510.com/xiazai/Application/other/30988.html

具体步骤

鼠标右键单击桌面空白处，打开git Gui Here，选择 Create New Repository（在本地创建一个仓库），如下

选择要托管项目本地存放位置(即本地新建仓库的位置)

点击Remote里面的Add

下一步之前需要github创建一个仓库用来存放

新建名字为department的仓库

继续add进入之后填写信息(成功后会提示success的绿色横线)

Rescan(刷新显示未缓存的demo文件)

Stage Changed(将未缓存文件缓存到)

Sign Off标志(右侧会提示一行信息)

Commit(提交命令)

上传到github(需要填写github用户密码)

查看github
